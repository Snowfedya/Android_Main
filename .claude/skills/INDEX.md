# Skill Index (Bridge to .agent/)

## Available Skills

All skills are located in `.agent/skills/`. Each skill has a `SKILL.md` file.

### Android Development
- `mobile-design` - Material Design, Android UI patterns, Purple Ban rule
- `clean-code` - SOLID, DRY, self-documenting code standards

### Architecture
- `architecture` - System design, patterns, trade-offs
- `api-patterns` - REST, GraphQL, RPC patterns
- `database-design` - Schema design, normalization, indexes

### Testing
- `testing-patterns` - Unit, integration, E2E tests
- `tdd-workflow` - Test-driven development
- `webapp-testing` - Web-specific testing

### Performance
- `performance-profiling` - Optimization techniques
- `vulnerability-scanner` - Security vulnerability detection

### Frontend
- `frontend-design` - UI/UX principles

### Backend

### Tools

## Usage

When an agent is invoked, it loads relevant skills from `.agent/skills/[skill-name]/SKILL.md`
