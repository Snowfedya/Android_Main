# PROJECT: Willpower Tracker

## Overview
Willpower Tracker is a modern Android application evolved from a series of lab assignments (Labs 1-7). It is designed to help users build and maintain habits using AI-driven insights and coaching.

## Goal
To transform a passive habit-tracking experience into an active personal growth journey by leveraging Jetpack Compose for a modern UI and the Gemini SDK for AI-powered coaching.

## Tech Stack
- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Architecture:** Clean Architecture (Data, Domain, UI) with MVI/MVVM
- **Dependency Injection:** Hilt
- **Local Database:** Room
- **Networking:** Retrofit
- **AI Engine:** Gemini SDK

## Context & Constraints
- Evolutionary step from Lab 1-7 (XML/Views to Compose).
- Focus on clean code, testability, and 2025 Android best practices.
- Offline-first capabilities for core habit tracking.
